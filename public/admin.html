🛠️ NUNI — Tableau de bord admin

Un seul endroit pour tout gérer : Pass, certifications, morceaux, utilisateurs.
🔒 Suspendre un compte
Bloque totalement la connexion de ce compte (même en pleine session), sans supprimer le compte ni son contenu.
🔓 Réactiver un compte suspendu
Redonne le droit de se connecter. Le Pass reste à réactiver séparément si besoin.
🗑️ Supprimer définitivement un compte
⚠️ Action irréversible. Supprime le compte et TOUT son contenu lié : morceaux, clips, abonnements, paiements, écoutes, abonnés.
🎁 Créer un nouveau code
🧹 Réinitialiser une certification (comptes de test)
⚙️ Taux de reversement
S'applique à tous les nouveaux calculs de reversement (pas rétroactif sur les paiements déjà enregistrés).
💸 Combien reverser à chaque artiste
Streams totaux = jamais touchés, toujours visibles publiquement. Streams de la période en cours = streams totaux moins tout ce qui a déjà été payé — recalculé à chaque fois, sans compteur à réinitialiser à la main.
